Сюда кладите каталоги как в Django MEDIA_ROOT:

  portfolio/images/*.jpg

Либо укажите другой путь на хосте через переменную окружения LEGACY_MEDIA_BIND
в .env (абсолютный или относительный путь к папке media старого проекта).

При старте API файлы копируются в постоянное хранилище uploads и создаются
записи в БД (коллекция, блог, hero/about в настройках).

Повторный импорт новых файлов при непустой коллекции: задайте один раз
FORCE_LEGACY_MEDIA_REIMPORT=true (осторожно: очистит коллекцию и блог).
